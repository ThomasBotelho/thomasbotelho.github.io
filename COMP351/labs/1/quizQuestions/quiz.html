COMP 351: Hard Coded Quiz

1) Please choose the correct result of this expression.

let user;
alert(user ?? "anonymous");

a) user
b) anonymous
c) null
d) "anonymous"

2) Which two values are multiplied together in this function?

let height = 0;
let width = 100;
let area = (height || 100) * (width || 50);
alert(area);

a) 100 * 100
b) 100 * 50
c) 0 * 50
d) 0 * 100

3) What is the value of x?

let x = (1 && 2 && 3) ?? 4;
alert(x);

a) 1
b) 2
c) 3
d) 4

4) Please choose the correct result of this expression.

let sum = (a, b) => a + b;
alert( sum(3, 4) );

a) 12
b) 3
c) 4
d) 7

5) Which code should be included within the empty brackets on the last line to achieve an integer output of 3?

let sum = (a, b) => {
   let result = a + b;
   return sum;
};
alert();

a) sum 1, 2
b) result 1, 2
c) sum(1, 2)
d) result(1, 2)

6) Please choose the correct result of this expression.

let double = n => n * 2;
alert( double(3) * 2);

a) 12
b) 6
c) 3
d) 2

Answers:

  1. b) anonymous
  2. a) 100 * 100
  3. b) 2
  4. d) 7
  5. c) sum(1, 2)
  6. a) 12
